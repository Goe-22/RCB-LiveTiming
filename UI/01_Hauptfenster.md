# Hauptfenster

## Ziel

Das Hauptfenster ist die zentrale Arbeitsoberfläche während des Trainings.

Alle wichtigen Informationen müssen ohne Scrollen sichtbar sein.

# Bedienung

- Klick auf Namen → Rundentabelle
- Klick auf Transpondernummer → Fahrer zuordnen
- Rechtsklick → Kontextmenü

---

# Farben

🟢 Grün = fährt

🟡 Gelb = Box

🔴 Rot = Decoderfehler

🔵 Blau = ausgewählter Fahrer

---

# Besonderheiten

Die Tabelle aktualisiert sich automatisch.

Der Benutzer muss nichts aktualisieren.

Alle Änderungen erfolgen ohne Unterbrechung der Zeitnahme.
